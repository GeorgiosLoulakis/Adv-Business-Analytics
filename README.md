# Stress-Aware Portfolio Optimizer with Reinforcement Learning

Final Project for the DTU course [42578 - Advanced Business Analytics](https://kurser.dtu.dk/course/42578).

---

## Overview

This project builds an intelligent portfolio management system that detects market stress and automatically repositions capital to minimize losses during crisis periods. The system combines machine learning, natural language processing, convex optimization, and reinforcement learning into a single pipeline.

The system was trained on 20 years of US market data (2005–2021) and tested on the out-of-sample period 2022–2026, covering the rate hike cycle, and the 2025 tariff shock.

---

## Structure

The entire analysis lives in a single notebook:

**`Porfolio_Optimasation_Notebook.ipynb`**

The notebook is organized into the following sections:

| Part | Description |
|------|-------------|
| Notebooks 01–03 | Data collection: prices, VIX/MOVE signals, NYT sentiment |
| Notebook 04:|
| Part A | Regime detection: classify each day as calm or stress |
| Part B | CVaR Optimizer: minimize worst-case loss per regime |
| Part C | Alternative benchmarks: Minimum Variance and Risk Parity |
| Part D | Backtest: 3-day rebalancing over out-of-sample period |
| Part E | Results: cumulative wealth, drawdowns, Sharpe |
| Part F | Resilience test: performance without sentiment signal |
| Part G | SHAP explainability: what drives the stress signal |
| Part H | RL Agent: PPO frozen and online adaptive agent |

---

## Methods

- **Stress signal** — Gradient Boosting quantile regression (5-day horizon)
- **Sentiment** — FinBERT scored on NYT financial headlines (2005–2026)
- **Regime detection** — combined z-score signal with 70th percentile threshold and 95th percentile shock override
- **CVaR Optimizer** — convex optimization via cvxpy with regime-dependent constraints
- **RL Agent** — PPO via Stable-Baselines3, trained on 2005–2021, tested on 2022–2026
- **Explainability** — SHAP values on the stress signal model
